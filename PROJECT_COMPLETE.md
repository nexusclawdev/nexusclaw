## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/nexusclawdev/nexusclaw.git
cd nexusclaw
pnpm install
pnpm build

# Setup wizard
nexusclaw onboard

# Start platform
nexusclaw gateway

# List all skills
nexusclaw skills list
```
