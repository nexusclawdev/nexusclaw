# Data Analysis Skill

This skill provides NexusClaw agents with data analysis capabilities.

## Usage
Agents can utilize this skill to:
- Parse CSV, JSON, and Excel files
- Generate statistical reports
- Create charts and visualizations
- Identify trends and patterns
- Export analysis results

## Examples
- "Analyze this sales data and show me the trends"
- "Create a chart showing monthly revenue"
- "Find correlations in this dataset"
