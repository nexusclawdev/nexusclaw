# Database Helper Skill

This skill provides NexusClaw agents with database management and optimization capabilities.

## Usage
Agents can utilize this skill to:
- Generate optimized SQL queries
- Optimize slow database queries
- Create safe migration scripts
- Design database schemas
- Debug query issues

## Examples
- "Generate a SQL query to find top 10 customers by revenue"
- "Optimize this slow query that takes 30 seconds"
- "Create a migration to add a new column to users table"
- "Design a database schema for an e-commerce platform"
- "Debug this SQL error: syntax error near WHERE"

## Capabilities
- SQL query generation for any database
- Query optimization with index recommendations
- Safe migration scripts (up and down)
- Database schema design with ER diagrams
- Query debugging and error resolution
- Performance analysis and tuning
