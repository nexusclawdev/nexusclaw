# Content Writer Skill

This skill provides NexusClaw agents with professional content creation capabilities.

## Usage
Agents can utilize this skill to:
- Write engaging blog posts and articles
- Create viral social media content
- Generate persuasive marketing copy
- Optimize content for SEO
- Adapt writing style for different audiences

## Examples
- "Write a 1000-word blog post about AI automation"
- "Create 5 Twitter posts for my product launch"
- "Generate marketing copy for my landing page"
- "Optimize this article for the keyword 'AI agents'"

## Prompts
The skill uses specialized prompts for:
- **Blog Post**: Creates structured, SEO-friendly articles
- **Social Media**: Generates platform-specific engaging posts
- **Marketing Copy**: Writes conversion-focused copy
- **SEO Optimize**: Improves content for search engines
