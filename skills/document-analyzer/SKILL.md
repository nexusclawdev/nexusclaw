# Document Analyzer Skill

This skill provides NexusClaw agents with advanced document analysis capabilities.

## Usage
Agents can utilize this skill to:
- Summarize long documents and PDFs
- Extract specific information and data
- Compare multiple documents
- Generate executive summaries
- Answer questions about document content

## Examples
- "Summarize this 50-page report in 5 key points"
- "Extract all dates and names from this contract"
- "Compare these 3 proposals and recommend the best one"
- "Create an executive summary for the board meeting"
- "What does this document say about pricing?"

## Capabilities
- Intelligent document summarization
- Structured data extraction
- Multi-document comparison
- Executive summary generation
- Question-answering from documents
- Citation and reference tracking
