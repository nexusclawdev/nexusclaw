# Code Reviewer Skill

This skill provides NexusClaw agents with comprehensive code review capabilities.

## Usage
Agents can utilize this skill to:
- Review code for bugs and logic errors
- Scan for security vulnerabilities
- Suggest performance improvements
- Enforce coding standards and best practices
- Generate detailed review comments

## Examples
- "Review this Python function for bugs and improvements"
- "Scan this code for security vulnerabilities"
- "Check if this code follows best practices"
- "Analyze the performance of this algorithm"

## Prompts
The skill uses specialized prompts for:
- **Full Review**: Comprehensive code analysis covering all aspects
- **Security Scan**: OWASP Top 10 vulnerability detection
- **Performance Check**: Identifies bottlenecks and optimization opportunities
- **Style Check**: Enforces coding standards and conventions
