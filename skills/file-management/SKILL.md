# File Management Skill

This skill provides NexusClaw agents with file system management capabilities.

## Usage
Agents can utilize this skill to:
- Search for files by name, type, or content
- Organize files into folders
- Rename files in bulk
- Create backups and archives
- Clean up duplicate files

## Examples
- "Find all PDF files modified in the last week"
- "Organize my downloads folder by file type"
- "Create a backup of my documents folder"
