# Web Automation Skill

This skill provides NexusClaw agents with browser automation capabilities using Playwright.

## Usage
Agents can utilize this skill to:
- Navigate websites and click elements
- Fill out forms automatically
- Take screenshots and recordings
- Extract data from web pages
- Automate repetitive browser tasks

## Examples
- "Take a screenshot of https://example.com"
- "Fill out the contact form on this website"
- "Extract all product prices from this page"
