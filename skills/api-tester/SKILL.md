# API Tester Skill

This skill provides NexusClaw agents with comprehensive API testing capabilities.

## Usage
Agents can utilize this skill to:
- Generate complete API test suites
- Create request examples in multiple languages
- Validate API responses against schemas
- Generate API documentation
- Design performance test plans

## Examples
- "Generate test cases for my REST API endpoint"
- "Create a curl request example for this API"
- "Validate this API response against the schema"
- "Generate OpenAPI documentation for my endpoints"
- "Design a load test plan for 10,000 concurrent users"

## Capabilities
- Comprehensive test case generation (happy path, edge cases, errors)
- Multi-language request examples (curl, JavaScript, Python)
- Response validation and schema checking
- API documentation generation (Markdown, OpenAPI, Postman)
- Performance testing strategies
