# Task Scheduler Skill

This skill provides NexusClaw agents with advanced task management capabilities.

## Usage
Agents can utilize this skill to:
- Create and organize tasks efficiently
- Schedule tasks based on priority and dependencies
- Set intelligent reminders
- Break down complex projects into manageable steps
- Optimize daily schedules

## Examples
- "Create a task to finish the report by Friday"
- "Break down my website redesign project into tasks"
- "Optimize my schedule for this week"
- "Set a smart reminder for my meeting preparation"

## Prompts
The skill uses AI-powered prompts for:
- **Create Task**: Generates structured tasks with optimal scheduling
- **Break Down Project**: Decomposes complex projects into actionable items
- **Schedule Optimization**: Creates efficient schedules based on constraints
- **Smart Reminders**: Sets context-aware notifications
