# System Monitor Skill

This skill provides NexusClaw agents with system monitoring and diagnostics capabilities.

## Usage
Agents can utilize this skill to:
- Monitor CPU, memory, and disk usage in real-time
- Track application performance metrics
- Set up intelligent alerts for anomalies
- Generate comprehensive health reports
- Diagnose and troubleshoot system issues

## Examples
- "Check my system health and report any issues"
- "Generate a performance report for the last week"
- "Set up an alert when CPU usage exceeds 80%"
- "Diagnose why my application is running slow"

## Prompts
The skill uses specialized prompts for:
- **Health Check**: Comprehensive system health analysis
- **Performance Report**: Detailed metrics and trend analysis
- **Alert Setup**: Configures intelligent monitoring alerts
- **Diagnose Issue**: Root cause analysis and remediation steps
