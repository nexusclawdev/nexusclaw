## 🚀 Enable GitHub Discussions (Manual Steps)

To enable contributions and community discussions:

### Step 1: Enable Discussions
1. Go to: https://github.com/nexusclawdev/nexusclaw/settings
2. Scroll to "Features" section
3. Check ✅ **Discussions**
4. Click "Set up discussions"

### Step 2: Configure Discussion Categories
Create these categories:
- 💡 **Ideas** - Feature requests and suggestions
- 🙋 **Q&A** - Questions and answers
- 📢 **Announcements** - Project updates
- 🎨 **Show and Tell** - Share your skills and projects
- 💬 **General** - General discussions

### Step 3: Add Topics/Tags
Go to: https://github.com/nexusclawdev/nexusclaw
Click ⚙️ next to "About" and add:
```
ai, agent, automation, browser-automation, typescript, nodejs,
playwright, security, multi-agent, llm, skills, productivity
```

### Step 4: Pin Important Issues
Pin these to the repository:
- Welcome/Getting Started issue
- Skill contribution guide
- Roadmap discussion

## 🤝 How Users Can Contribute Skills

Users can now easily contribute by:

1. **Fork the repository**
2. **Create a new skill** in `skills/your-skill-name/`
3. **Add two files:**
   - `skill.json` - Metadata and AI prompts
   - `SKILL.md` - Documentation
4. **Submit a Pull Request**

Example skills to reference:
- `skills/email-assistant/` - Complex prompts
- `skills/code-reviewer/` - Security-focused
- `skills/content-writer/` - Marketing-focused
